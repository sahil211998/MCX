Trade with logic, not emotion — every signal backed by data, pattern, and discipline.

Only six contracts: ALUMINI, ZINCMINI, SILVERM, CRUDEOILM, NATURALGAS, NATGASMINI. Does not guarantee profit. Click to load full history, intraday proxy (default 30m, optional 15m), long-term proxy (default 4h), swing targets; call time is stored. Intraday timing uses a Yahoo proxy; intraday prices in the app are scaled to MCX INR using live LTP or last MCX daily close.

How this maps to institutional-style signal tools

Many prop and futures desks layer VWAP, relative volume, and prior-day (PD) high/low/close with platforms that show order flow (footprint, delta, DOM). Examples in that space include vendor order-flow analytics and multi-confluence chart suites (often paired with data terminals). This project does not subscribe to paid institutional feeds or MCX L2/L3 tape—it adds the same family of price/volume structure metrics we can compute from public bhav, watch LTP, and Yahoo proxy bars (intraday + long-term), plus your existing EMA/RSI/ATR/MACD/Bollinger confluence.
